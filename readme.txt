== Description ==
Sæt nemt og sikkert opkalds- og emailtracking op på din WordPress hjemmeside. Det primære formål med plugin’et er at automatisere opsætningen af trackingværktøjer. Når plugin’et er installeret kan du se statistik om dine henvendelser i SpotOn Live.

== Installation ==
Opkaldstracking
1. Tilføj SpotOn Live API-nøgle. Tryk Gem.
2. Klik på knappen Opkaldstracking > Aktiver

Emailtracking
1. Et krav for at bruge emailtracking er at din WordPress hjemmeside bruger email-formularer fra Gravity Forms (http://www.gravityforms.com/).
2. Tilføj SpotOn Live API-nøgle. Tryk Gem.
3. Vælg Emailtracking > Aktiveret.
